Para utilizar o sistema:

  • Login:

Realize o login com o seu usuário cadastrado para poder acessar o sistema

- No campo "Usuário" preencha corretamente o seu usuário cadastrado

- No campo "Senha" preencha corretamente a sua senha cadastrada

- Clique no botão "Entrar" para realizar o Login

  • Cadastro de produtos:

Na tela inicial do sistema, é possivel observar um botão "Novo produto"

- Ao clicar no botão, a tela de cadastro de produtos se abrirá.

- Nessa nova tela, é possivel preencher os dados do novo produto a ser cadastrado.

- Informe o Nome, a Descrição, a Quantidade, o Preço, o Lote e a imagem do novo produto

- Com as informações preenchidas, clique no botão "Salvar"

- Você será redirecionado para a tela de Listagem de produtos e poderá vizualizar o novo produto cadastrado

* Caso desistir de cadastrar um novo produto, clique no botão "Cancelar"

  • Listagem de produtos:

Na tela inicial do sistema, é possivel observar todos os produtos que foram cadastrados no sistema

- É possivel vizualizar suas informações como: Id, Nome, Descrição, Quantidade, Preço, Lote e a sua imagem
